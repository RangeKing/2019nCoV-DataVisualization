# 2019nCoV-DataVisualization

### 📖简介

> 一个中国新型冠状病毒疫情数据可视化的例子

2020年1月，中国新型冠状病毒疫情发酵。看到网上制作的疫情地图质量参差不齐，故利用疫情数据绘制了一些图表。主要目的为个人python数据可视化、Git和markdown等学习之用（~~没什么基础，献丑了~~）。顺道收集了一些个人认为比较优秀的案例做分享。
